About

This tool has been designed to illustrate the effects of CH4, CO2 and cloud on earth's radiation balance. It was produced by the Faculty of Engineering and Physical Sciences eLearning team in collaboration with the Centre for Atmospheric Science both at the the University of Manchester.

It uses modern web technologies which allow it to work on both small screen mobile devices and large desktop computers. It has been tested with modern versions of the following browsers:

  • Firefox (Windows)
  • Chrome (Windows)
  • Safari (Mac)
  • Safari (iOS)
  • Chrome for Android (Android)

Unfortunately, this simulation does not work well on Windows Internet Explorer (despite much effort) due to bugs with this browser. If you experience any problems using the tool, we recommend in the first instance you reload the page and if you still experience problems, try using a recent version of a different web browser.
